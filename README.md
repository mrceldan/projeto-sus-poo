# projeto-sus-poo
Projeto desenvolvido na disciplina de Programação Orientada a Objetos, com objetivo de aplicar de forma prática conceitos como classes, objetos, encapsulamento, herança e polimorfismo. O trabalho foi construído em sala, com orientação do professor e colaboração da turma, reforçando a lógica estruturada e o pensamento no desenvolvimento de software.
